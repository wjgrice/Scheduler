Author: William Grice
Contact Information: wgrice1@wgu.edu
Student Application Version: 1.0
Date: February 22, 2023
IDE: IntelliJ IDEA 2022.3.2 (Community Edition)
JDK: Java SE Corretto-17.0.6
JavaFX: JavaFX-SDK-17.0.2-ea
MySQL Connector Driver Version: mysql-connector-java-8.0.32

Purpose:
The Appointment Management System is a Java application that allows users to manage customer information and schedule
appointments.

Directions for Running the Program:
    1. Clone the repository or download the project files.
    2. Open the project in IntelliJ.
    3. Ensure that the MySQL Connector driver is installed and properly configured.
    4. Run the application by clicking the 'Run' button in the IDE.

Additional Report:
One additional report that can be run in the Appointment Management System is a report of all with a time frame filter
and a total of all minutes for the filtered period.
